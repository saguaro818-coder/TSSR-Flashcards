# TSSR-Flashcards
Flashcards orientées mises en situation pour Technicien Supérieur Systèmes et Réseaux. Cas pratiques couvrant support utilisateur, serveurs Windows/Linux, réseau IP, virtualisation, scripts, sauvegardes, déploiement et sécurité. Format question/réponse centré sur diagnostic, décision et action en environnement professionnel.
